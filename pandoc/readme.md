# Pandoc konverterio naudojimas

## Paleidimas

Argumentai

```bash
./pandoc-bun.mjs --opt=įvesties-formatas/išvesties-formatas failo-pavadinimas-be-plėtinio
```

Pavyzdys:

```bash
./pandoc-bun.mjs --opt=markdown/asciidoc readme
```
