## Bun

### Diegimas

```bash
curl -fsSL https://bun.sh/install | bash
bun --version
```

### Paleistis

```bash
bun run js_sys-upgrade-bun.mjs
```

arba

```bash
./js js_sys-upgrade-bun.mjs
```
